# Python 기초

Python은 문법이 간결하고 다양한 라이브러리를 제공하는 프로그래밍 언어입니다. 데이터 분석, 웹 개발, 인공지능, 자동화 등 다양한 분야에서 사용됩니다.

## 변수

변수는 데이터를 저장하기 위한 이름입니다. Python에서는 변수의 자료형을 명시하지 않아도 값을 할당할 수 있습니다.

## 조건문

조건문은 특정 조건에 따라 다른 코드를 실행하기 위해 사용합니다. Python에서는 if, elif, else 키워드를 사용합니다.

## 반복문

반복문은 같은 작업을 여러 번 실행할 때 사용합니다. Python에서는 for문과 while문을 사용할 수 있습니다.

## 함수

함수는 특정 기능을 수행하는 코드 블록입니다. 함수를 사용하면 중복 코드를 줄이고 프로그램을 더 구조적으로 작성할 수 있습니다.

## 리스트

리스트는 여러 값을 순서대로 저장할 수 있는 자료형입니다. 리스트는 인덱스를 사용하여 각 요소에 접근할 수 있습니다.
