# 머신러닝 기초

머신러닝은 데이터를 기반으로 컴퓨터가 규칙이나 패턴을 학습하도록 하는 인공지능 분야입니다. 사람이 모든 규칙을 직접 작성하지 않아도, 모델은 데이터에서 특징을 찾아 예측이나 분류를 수행할 수 있습니다.

## 지도학습

지도학습은 입력 데이터와 정답 레이블을 함께 사용하여 모델을 학습시키는 방법입니다. 예를 들어 이메일이 스팸인지 아닌지 분류하거나, 주택 가격을 예측하는 문제가 지도학습에 해당합니다.

지도학습의 대표적인 문제 유형은 분류와 회귀입니다. 분류는 데이터를 정해진 범주 중 하나로 나누는 작업이고, 회귀는 연속적인 숫자 값을 예측하는 작업입니다.

## 비지도학습

비지도학습은 정답 레이블 없이 데이터의 구조나 패턴을 찾는 학습 방법입니다. 대표적인 예로 군집화와 차원 축소가 있습니다.

군집화는 비슷한 특성을 가진 데이터를 그룹으로 묶는 방법입니다. 차원 축소는 많은 특징을 가진 데이터를 더 적은 수의 특징으로 압축하여 분석하기 쉽게 만드는 방법입니다.

## 과적합

과적합은 모델이 학습 데이터에 지나치게 맞춰져 새로운 데이터에 대한 성능이 낮아지는 현상입니다. 이를 줄이기 위해 검증 데이터 사용, 정규화, 데이터 증강 등의 방법을 사용할 수 있습니다.
